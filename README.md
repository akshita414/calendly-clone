# 📅 Calendly Clone

A full-stack scheduling application inspired by Calendly that allows users to create events and book meetings seamlessly.

---

## 🚀 Features

### Event Management
- Create event types (name, duration, slug)
- Unique public booking links
- Delete events

### Booking System
- Select date and time slots
- Enter name and email
- Booking confirmation page
- Prevent double booking

### Availability
- Weekday availability (Mon–Fri)
- Fixed time slots

### Meetings Dashboard
- View upcoming meetings
- View past meetings
- Cancel meetings

---

## 🛠 Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: JSON file
